# Weather Prediction Prototype

Small Python notebook scaffold for a 1-day weather forecasting pet project.

## Contents

- `notebooks/1day_weather_forecast_scaffold.ipynb`: starter notebook for pulling daily weather data, building simple lag features, and comparing a baseline with a first regression model.

## Getting Started

Install the starter dependencies:

```bash
pip install pandas numpy matplotlib scikit-learn meteostat
```

Open the notebook and run the cells top to bottom.
